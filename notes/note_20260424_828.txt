INSIGHTS AND NOTES

one potential gotcha when using backtrader is to make sure to set up the data feed correctly, as incorrect data feed settings can result in incorrect backtesting results. another interesting thing i noticed is that backtrader's default commission and slippage settings can have a significant impact on backtest results, so it's essential to adjust them according to the actual trading conditions. another tip i learned is to keep the backtesting timeframe short, especially when dealing with high-capital strategies, as longer backtesting periods can result in overfitting.
